## Tarefa: Liquid Glass Dark Design System — Speed Extension

### Descrição
Aplicado o design system completo "Liquid Glass Dark" às extensões Starble Speed (panel.html e popup.html).

### Implementado
- **Panel.html**: Layout com nav lateral 54px + 3 telas (Home, Chat, Settings) com navegação por botões com indicador ativo
- **Tokens CSS**: Sistema completo de cores com variáveis whitelabel (`--wl-primary`) substituíveis dinamicamente
- **Glass effects**: `backdrop-filter: blur(40px) saturate(220%)`, bordas 0.5px, inset highlights
- **Chat UI**: Bolhas com gradiente primário (user) e glass (AI), typing dots animados, suggestion chips, mode selector
- **Home**: Hero card com glow, status rows com dots animados, QA grid 2×2 com spring hover, usage bar, booster card
- **Popup**: Mesmos tokens e componentes glass aplicados, cards de licença/ativação/status
- **Animações**: `cubic-bezier(.34,1.56,.64,1)` spring, `panel-in` entrada, `pulse-glow` para dots ativos
- **Scrollbar**: Invisível em scroll areas, 2px ultrafina no chat
