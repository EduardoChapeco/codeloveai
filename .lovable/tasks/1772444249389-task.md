## Tarefa: cirius new design

### Descrição
Auditoria completa do Cirius Editor — verificação de implementação do design system Islands, Drawers e Bubbles.

### Resultado
✅ **Totalmente implementado conforme especificação.**

**Componentes verificados (15/15):**
- `CiriusEditor.tsx` — Root com estado global, keyboard shortcuts (⌘K), realtime, drawers same-side logic
- `PreviewArea.tsx` — 3 frame modes (desktop/tablet/mobile) + empty state com grid + ícone
- `IslandLeft.tsx` — Logo + project selector + domain + SEO buttons
- `IslandCenter.tsx` — Frame selector (Desktop/Tablet/Mobile) com `.frm-btn`
- `IslandRight.tsx` — Status pill animado + icon buttons + Publicar
- `BottomIsland.tsx` — Mode strip (Build/Tarefa/Encadeado/Review/Debug/BLE) + textarea auto-resize + toolbar + queue pill
- `DomainIsland.tsx` — Toggle island com input monospace + salvar
- `TaskBubbles.tsx` — Spawn dinâmico, timer, steps com dots, progress bar, auto-dismiss
- `CmdPanel.tsx` — ⌘K overlay com modos Chat/Code, file tabs, syntax highlighting
- `DrawerDeploy.tsx` — GitHub toggle + Vercel/Netlify/Supabase + Deploy Agora
- `DrawerFiles.tsx` — Árvore de arquivos com grouping + Baixar todos
- `DrawerSEO.tsx` — Score display + campos (título, meta, OG, canonical) + Salvar
- `DrawerBuild.tsx` — Orquestrador com fases, thinking indicator, task cards expandíveis, files table
- `DrawerChain.tsx` — Lista encadeada com chain-num dots + Add Task + Run All
- `EditorToasts.tsx` — Toast notifications com success/info + auto-dismiss

**CSS verificado:**
- `cirius-editor.css` — 752 linhas com todos os tokens do spec
- Glass buttons (`.gl`) com shimmer hover, 6 tamanhos, 5 variantes de cor
- Chips (`.ce-chip`) com 6 cores
- Side drawers com animação scale+translateY, hidden/visible states
- Task bubbles com bubble-in/bubble-out animations
- CMD panel com cmd-in animation
- Todas as variáveis CSS (`--bg-0` a `--bg-5`, `--b0` a `--b3`, `--text-*`, `--ease-spring`, `--r2` a `--rF`, cores accent) definidas no `index.css`
