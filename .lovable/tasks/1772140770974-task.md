## Tarefa: Fix acesso admin/tenant

### Problema
Tenant admins/owners não conseguiam salvar configurações nem revogar licenças no painel `/admin/tenant` porque faltavam políticas RLS de UPDATE nas tabelas `tenants` e `licenses`.

### Correção
- Adicionada política RLS `"Tenant admins update own tenant"` na tabela `tenants` (FOR UPDATE) usando `is_tenant_admin(auth.uid(), id)`
- Adicionada política RLS `"Tenant admins update own tenant licenses"` na tabela `licenses` (FOR UPDATE) usando `is_tenant_admin(auth.uid(), tenant_id)`

*Criado via security fix*
