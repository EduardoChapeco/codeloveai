## Tarefa: Melhorias Brain — Estabilidade e UX

### Descrição
Correções de estabilidade do Brain, detecção de troca de conta, UX do chat e estados de processamento.

### Correções Aplicadas
1. **Account mismatch auto-recovery**: Setup e send agora verificam se o projeto Brain é acessível na conta Lovable atual. Se não for (403/404), deleta o registro antigo e recria automaticamente.
2. **Stale lock cleanup**: Locks de criação com mais de 2 minutos são automaticamente liberados, evitando travamentos permanentes.
3. **Processing UX**: Substituída a mensagem estática "Processando (até 90s)" por fases animadas progressivas (Conectando → Analisando → Processando → Gerando → Refinando → Finalizando) com barra de progresso e tempo decorrido.
4. **Chat design fix**: Corrigidos problemas de overflow em blocos de código, melhorada a responsividade mobile, separado seletor de brain type para mobile.
5. **Error recovery**: Mensagens com falha agora mostram o erro específico e botão "Tentar novamente" que preenche o campo de input com a mensagem original.
6. **getBrainRaw helper**: Nova função para ler o estado bruto do brain (incluindo status "creating") para decisões mais precisas.

- `src/pages/Brain.tsx` — UX completa refeita com ProcessingIndicator, retry, design fixes
- `supabase/functions/loveai-brain/index.ts` — v8.2 com account mismatch detection
