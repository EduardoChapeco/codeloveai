## Tarefa: Correção e Melhoria de Edge Functions — Brain Flow

### Implementado

1. **Bootstrap PRD Aprimorado**: O prompt inicial do Ghost Create agora usa formato PRD (Product Requirements Document) com persona expert completa (PhD/Sênior), incluindo credenciais, foco e protocolo de resposta em .md/.json/.html.

2. **Audit Prompts Sequenciais**: Após o bootstrap, 5 prompts de auditoria são injetados em background a cada 60 segundos para verificar estrutura, criar templates de resposta, manifesto de capacidades, auto-teste e verificação final de prontidão.

3. **Context Injection**: O bootstrap agora injeta contexto real (SUPABASE_URL, ANON_KEY, userId) no projeto via config.json para comunicação call-home.

4. **Ghost Cancel mantido**: Cancelamento após criação em ~1s (já existente), seguido de 8s de estabilização antes do bootstrap PRD.

5. **Fluxo completo**: Create → Cancel (1s) → Wait (8s) → Bootstrap PRD → Wait (60s) → 5x Audit Prompts (60s each)
