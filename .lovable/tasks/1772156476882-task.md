## Tarefa: Sidebar sem scroll visível, background seamless, sem linhas divisórias

### Descrição
O sidebar (tem um scroll visível) não poderia ter scroll visível. O sidebar deve ter scroll vertical mas sem barra visível. O background do sidebar e da página deve ser a mesma cor sempre, sem distinção. O sistema deve ser sem linhas/line/elementos dividindo grids/layouts.

### Resolução
- Scrollbar global configurada para invisível (`scrollbar-width: none`, `::-webkit-scrollbar { width: 0 }`)
- Classe `.scroll-on-hover` adicionada para scroll com barra que aparece apenas no hover
- Background do sidebar (`clf-glass-sidebar`) alterado para transparente, sem `border-right`
- Removidas todas as `border-t`, `border-b` e `borderBottom/borderRight/borderTop` inline dos componentes:
  - `AppSidebar.tsx` — footer border removido
  - `Brain.tsx` — sidebar border, header borders, input bar border, response divider removidos
- Background do sidebar do Brain alterado de `liquid-glass-bg` para `transparent`
